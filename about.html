About EventHub

Your trusted Event Management Platform

Who We Are

EventHub is a modern Event Management System designed to simplify event discovery, online booking, and event management. Users can explore different categories of events, book tickets instantly, and manage their bookings through an easy-to-use interface.

This project is developed using HTML, CSS, JavaScript, Spring Boot, and MySQL as a full-stack web application.
